## 🎯 **Sobre o Projeto**

O **CalangoFluxRuralLabs** é uma iniciativa revolucionária que nasce no coração do meio rural brasileiro para democratizar o acesso ao letramento digital. Nosso Co-Lab combina:

- � **Saberes Tradicionais** + 💻 **Tecnologia Digital**
- 🤝 **Aprendizado Horizontal** + 🌱 **Crescimento Coletivo**
- 🏡 **Raízes Rurais** + 🌐 **Conexão Global**

### 🎪 **Primeiro Ciclo - 2º Semestre 2025**
- ✅ **100% Gratuito** - Mantido exclusivamente por doações
- 🎁 **Notebook para Casa** - Cada participante leva o equipamento (85% presença + 90% tarefas)
- 🌟 **Para Todas as Idades** - Infantil, jovem, adulto, sênior
- 🔄 **Sem Fins Lucrativos** - Movimento de partilha e autonomia
